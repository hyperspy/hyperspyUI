class hyperspyui.bindinglist.BindingList(target=None, *args, **kwargs)ÔÉÅ

Bases: list

A list that has been extended to sync other lists or collections to changes in its contents. By custom targets, it can also be used to trigger events on addition/removal. Only append and remove actions are required, as extend, insert and pop can be inferred (insert loses order however), but for reasons of speed, it is recommended to supply all if they are available.
