hyperspyui.uiprogressbar._wrap(*args, **kwargs)Â¶

Replacement function for hyperspy.external.progressbar.progressbar(). Causes a UIProgressBar() to be made, which the MainWindow can connect to in order to create a progress indicator. It is important that the connection is made with QtCore.Signals, as they are thread aware, and the signal is processed on the GUI main event loop, i.e. the main thread. This is necessary as all UI operations have to happen on the main thread, and the hyperspy processing might be pushed to a worker thread âthreaded.pyâ.
